 CRUD de Alunos

## Descrição
Este projeto é uma aplicação web de exemplo para realizar operações de **CRUD** (Criar, Ler, Atualizar e Deletar) de alunos.  
O sistema consome uma **API pública** e foi desenvolvido como prática de estudo em **HTML, CSS (Bootstrap) e JavaScript puro**.

---

## Funcionalidades
- Listar alunos cadastrados
- Cadastrar novo aluno
- Editar informações de um aluno
- Excluir aluno da lista
- Validação de campos do formulário
